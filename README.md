# 🔐 Password Generator

This is a simple Password Generator web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Generate random secure passwords
- Choose password length
- Include uppercase, numbers, and symbols
- Check password strength
- Copy password to clipboard

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How it works
User enters password length and selects options. The app generates a random password using JavaScript logic and displays it instantly.
